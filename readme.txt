== Description ==

The Zendesk extension for Ninja Forms allows you to create a new ticket in Zendesk when a form is submitted.
The 'Ninja Forms - Zendesk' extension includes supports for the Zendesk end-user custom fields.

== Requirements ==

WordPress 5.4+
NinjaForms 3.0+

== Installation ==

This section describes how to install the plugin and get it working.

1. Download the repository as .zip file
2. Go to: WordPress Admin > Plugins > Add New and Upload Plugin with the file you downloaded with Choose File.
3. Install Now and Activate the extension.
4. Visit the 'Ninja Forms' menu item in your admin sidebar
5. When you create a form, you can now add 'Zendesk' actions on the 'Emails & Actions' page.
